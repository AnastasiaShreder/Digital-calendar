 - 3 веб-страницы
   1) Авторизация
      * База данных логинов и паролей пользователей 
      * Кнопка "Запомнить меня"
      * Возможность выбора региона
   2) Основная страница
      * Календарь (месяц/неделя)
      * Меню
        1. Общий календарь - отображение всех проектов с помощью иконок
           * Фильтр (по регионам, внешние/собственные мероприятия, по группам мероприятий, отображение праздников);
           * Щелкаем на день - отображение списка мероприятий на день(если их несколько);
           * Отображение текущего времени
        2. Мой календарь - отображение собственных проектов на своем календаре
           * Возможность добавить проект и задачу;
              - окно создания задачи/редактирования
              - окно создания проекта/редактирования
           * Щелкаем на день - отображение всех задач на день, сгруппированных по проектам (используем разные цвета);
           * Фильтр (по группам мероприятий, отображение праздников) 
           * Возможность отображать только проект и относящиеся к нему задачи
        3. Список задач (новая страница)
           * Отображение на неделю, месяц, день
           * Приоритетность задач (задается при создании и выделяется разными цветами)
           * Возможность выделить задачу, как отмеченную;
    3) Окно создания задачи
       * Название задачи
       * Дедлайн
       * Мероприятие, к которому создается задача
       * Исполнитель
       * Приоритетность задачи
       * Дата создания задачи(автоматически вбиывается)
    4) Окно создания проекта 
       * Название проекта
       * Дата, время проведения, дедлайн
       * Место проведения
       * Окно для создания первой задачи (опционально)
       * Руководитель и участники проекта
       * Группа проекта (по умолчанию "Общая")
       * Комментарий к проекту
       * Автор проекта(создатель, автоматически вводится имя текущего пользователя)
       * Дата начала проекта
